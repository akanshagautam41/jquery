Q1. What is jQuery?

Ans . jQuery is a fast, small, and feature-rich JavaScript library. It simplifies tasks like HTML document traversal and manipulation, event handling, animation, and AJAX interaction. jQuery's syntax is designed to make it easier to navigate a document, select DOM elements, create animations, handle events, and develop AJAX applications. It's widely used in web development to enhance user interface interactivity and streamline coding processes.

Q2 .How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery , JQuery Animation?

To apply CSS using jQuery, add or remove classes, and create animations, you can use jQuery methods and functions. Here's a breakdown of how to achieve each of these tasks:

1. Applying CSS Using jQuery:

You can apply CSS styles to HTML elements using jQuery's .css() method. This method allows you to set one or more CSS properties and their values for selected elements.

2. Adding and Removing Classes Using jQuery:

To add or remove classes from HTML elements using jQuery, you can use the .addClass() and .removeClass() methods respectively.

3. jQuery Animation:

jQuery provides several methods for creating animations, such as .hide(), .show(), .fadeIn(), .fadeOut(), .slideUp(), .slideDown(), .animate(), etc. These methods allow you to animate the visibility, position, and style of HTML elements.
